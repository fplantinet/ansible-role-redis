# Ansible Role Redis

Ansible role to install redis server.

## Requirements

* Ansible version 2.*

## Role Variables

You can override all the variables in `defaults/main.yml`.

## Example Playbook

    - hosts: servers
      roles:
         - { role: monsieurbiz.redis }
